# Comparador de Divisas

Este proyecto es un sitio web que proporciona una tabla comparativa del valor de diversas divisas en relación con una moneda seleccionada. También cuenta con una barra de búsqueda que permite elegir la moneda de referencia para la comparación y ofrece opciones de ordenamiento por nombre de moneda y por valor.

## Características

- Tabla comparativa de valores de divisas.
- Barra de búsqueda para seleccionar la moneda de referencia.
- Ordenamiento de la tabla por nombre de moneda y valor.

## Tecnologías utilizadas

- React
- Redux Toolkit
- Tailwind CSS

## Instalación

1. Clona este repositorio: `git clone https://github.com/toleZ/ForexTrack.git`
2. Instala las dependencias: `npm install`
3. Inicia la aplicación: `npm start`
